# ThetaChart 📊

A tool to analyze stock options premiums and detect market strength/weakness.

🔗 **Live Demo:** [thetachart.com](https://thetachart.com)

## Features
- View real-time options chain
- Filter by volume
- Calculate Greeks (Delta, Gamma, Theta, Vega)
- Implied volatility analysis

## Tech Stack
- Python
- Streamlit
- yfinance
- Pandas

## How to Run
pip install -r requirements.txt
streamlit run src/main.py
